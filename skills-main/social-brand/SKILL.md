# Social Brand Guidelines

## Overview

This skill provides brand guidelines for AUTOMATION Creators, ensuring consistent visual identity and brand voice across all social media content, marketing materials, and digital assets.

## Brand Identity

### Brand Name & Tagline
- **Brand Name:** AUTOMATION Creators
- **Tagline:** Profitability on Autopilot
- **Voice:** Professional, tech-savvy, results-oriented, empowering

### Brand Promise
Focus on automation solutions that drive profitability and efficiency. Content should emphasize practical value, expertise in automation, and measurable business outcomes.

## Visual Guidelines

### Color Palette

**Primary Colors:**
- **Cyber Cyan:** #00D9D9 (primary accent color)
  - Use for: CTAs, highlights, interactive elements, key headings
  - RGB: (0, 217, 217)
  
- **Deep Black:** #000000 (primary background)
  - Use for: backgrounds, bold statements, contrast
  - RGB: (0, 0, 0)

**Supporting Colors:**
- **White:** #FFFFFF (text and contrast)
- **Dark Gray:** #1A1A1A (secondary backgrounds)
- **Muted Cyan:** #008B8B (secondary accents, hover states)

**Color Usage Rules:**
- Maintain high contrast ratios (cyan on black, white on black)
- Use cyan sparingly as accent color to maintain impact
- Avoid using cyan as background color
- Black should dominate the visual space for premium feel

### Typography

**Headline Fonts:**
- Use bold, condensed sans-serif fonts for major headlines
- Style: All caps for maximum impact
- Weight: Extra bold (800-900)
- Characteristics: Tight letter-spacing, strong presence
- Example usage: "ULTIMATE PROMPTS", "AUTOMATION", main value propositions

**Body Text:**
- Clean, modern sans-serif
- Weight: Regular (400) for body, Medium (500) for subheadings
- Maintain excellent readability against dark backgrounds
- Use white or light gray for body text

**Accent Text:**
- Cyan color for key phrases and CTAs
- Can be used in both uppercase and mixed case
- Reserve for important messaging and calls-to-action

### Visual Elements

**Background Patterns:**
- Dotted grid pattern creates tech-forward aesthetic
- Small, evenly-spaced dots in subtle gray (#333333)
- Creates depth without overwhelming content
- Maintain consistency across all branded materials

**Iconography:**
- Robot/bot mascot represents automation
- Clean, geometric line style
- Cyan and white color scheme
- Friendly yet professional appearance
- Can be used as brand identifier

**Layout Principles:**
- Center-aligned layouts for social media content
- Generous white space (or black space) around key elements
- Clear visual hierarchy with oversized headlines
- CTAs should be prominent and unmissable

### Button & CTA Design

**Style:**
- Rounded rectangles (border-radius: 25-30px for pill shape)
- Cyan background (#00D9D9)
- Black text for maximum contrast
- Uppercase or title case text
- Arrow symbols (â†’) to indicate action

**States:**
- Default: Solid cyan fill
- Hover: Slightly lighter cyan or subtle glow effect
- Active: Pressed state with darker cyan

**Sizing:**
- Large, finger-friendly dimensions for mobile
- Minimum height: 50px
- Generous padding: 20-30px horizontal

## Content Guidelines

### Messaging Tone
- **Professional but approachable:** Expert guidance without jargon
- **Action-oriented:** Focus on outcomes and results
- **Empowering:** Position audience as capable of achieving automation success
- **Value-driven:** Always emphasize ROI and practical benefits

### Content Patterns
- Lead with value proposition
- Use power words: Ultimate, Guide, Mastery, System, Blueprint
- Create curiosity gaps that encourage engagement
- Include clear calls-to-action
- Focus on transformation and results

### Social Media Applications

**Instagram/Facebook Posts:**
- Square format (1080x1080px)
- Bold headline at top
- Visual hierarchy with size contrast
- CTA button at bottom
- Brand elements (robot icon, tagline) in appropriate corners

**Stories/Reels:**
- Vertical format (1080x1920px)
- Swipeable content indicated with arrows or "SWIPE" text
- Keep key elements in safe zones (avoid top/bottom edges)
- Use cyan accents to guide eye movement

**LinkedIn Posts:**
- Professional tone while maintaining brand style
- Can use slightly more formal language
- Emphasize business outcomes and ROI
- Include data points and statistics when possible

**Lead Magnets/Guides:**
- Clear, benefit-driven titles
- "GUIDE TO" format works well
- Use dotted background pattern
- Include website URL at bottom (automationcreators.beehiiv.com or similar)
- Maintain consistent layout across series

## Application Examples

### When creating social media content:
1. Start with black background
2. Apply dotted grid pattern (subtle)
3. Position robot icon in corner
4. Use oversized white text for main headline
5. Add "GUIDE TO" or similar in cyan pill shape
6. Include tagline "Profitability on Autopilot" near brand name
7. Place CTA button in cyan at bottom
8. Add website URL in small white text

### When designing presentations:
1. Use black slides with cyan accents
2. Apply dotted pattern to backgrounds
3. Use bold, oversized headlines
4. Limit text per slide (high impact)
5. Include brand elements consistently
6. Maintain generous spacing

### When creating artifacts/documents:
1. Apply color scheme consistently
2. Use appropriate typography hierarchy
3. Include brand identity elements
4. Maintain professional yet bold aesthetic
5. Ensure high contrast for readability
